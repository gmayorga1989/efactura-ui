# Documentacion interna frontend

Esta carpeta contiene los estandares tecnicos y flujos de implementacion del frontend de eFactura EC.

## Documentos

- [Estandares frontend y flujos](./frontend-estandares-y-flujos.md)

## Regla general

Todo cambio funcional o nueva pantalla debe actualizar la documentacion cuando afecte:

- estructura visual o UX estandar,
- i18n runtime,
- menus o rutas,
- grids Tabulator,
- modales y confirmaciones,
- flujos de sesion, avatar o presencia,
- contratos esperados con backend.
